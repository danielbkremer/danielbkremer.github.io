# Daniel Kremer — Personal Site (v0)

Static HTML/CSS site intended for GitHub Pages.

## Deploy on GitHub Pages
1. Repo → Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: `main` and folder `/ (root)`

## Files
- `index.html` — homepage + press-facing summary
- `filmography.html` — curated filmography + link to IMDb
- `books.html` — books list + links
- `commentaries.html` — selected home media work (table you can expand)
- `assets/style.css` — site styling
- `assets/daniel-kremer.jpg` — hero image

## Edit contact
Open `index.html` and replace:
`Add email / press contact here`

## Add content
- Filmography: edit the table in `filmography.html`
- Commentaries: add rows in `commentaries.html`
